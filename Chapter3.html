Chapter 3: Operators and Control Statements - Summary

1. Operators Overview

  • Operands are the data items acted on by operators.
  • Example: age = currentYear - birthYear + 1 → operands: age, currentYear, birthYear, 1.

2. Arithmetic Operators

  • / — Division
  • % — Remainder
  • Increment / Decrement examples:
let a = 10;
console.log(a++); // prints 10, then a becomes 11
console.log(++a); // a becomes 12, then prints 12

3. Assignment Operators

  • Combine arithmetic and assignment for shorthand:
y += x;  // y = y + x
y -= x;
y *= x;
y /= x;
y %= x;

4. String Concatenation Operator

  • + connects strings and adds numbers.
  • If one operand is a string, concatenation occurs:
alert("Hello " + user + "!");

5. Comparison Operators

  • Return true or false based on comparison.
OperatorMeaning
==, !=Equal / not equal (type conversion allowed)
===, !==Strict equal / not equal (no type conversion)
<, <=, >, >=Comparison operators

6. Logical Operators

  • && — AND
  • || — OR
  • ! — NOT
  • Commonly used for combining conditional statements.

7. Conditional Statements

if (age < 20) {
  alert("You are a minor.");
} else if (age >= 20) {
  alert("Adult");
} else {
  alert("Not adult");
}

8. Example: Multiple of 3

let num = prompt("Enter a number:");
if (num !== null) {
  num = parseInt(num);
  if (num % 3 === 0) alert("Multiple of 3");
  else alert("Not a multiple of 3");
}

9. Truthy and Falsy Values

  • Falsy values: 0, "", null, undefined, NaN.
  • All other values are truthy.

10. Ternary Operator

let small = (num1 < num2) ? num1 : num2;

11. Even / Odd Checker

let n = prompt("Enter a number:");
if (n !== null) {
  n = parseInt(n);
  (n % 2 === 0) ? alert("Even") : alert("Odd");
}

12. switch Statement

let subject = prompt("Choose: 1-HTML, 2-CSS, 3-JS");
switch (subject) {
  case "1": alert("HTML chosen"); break;
  case "2": alert("CSS chosen"); break;
  case "3": alert("JavaScript chosen"); break;
  default: alert("Invalid input");
}

13. Multiple Conditions

  • || — true if either condition is true
  • && — true if both are true
  • ! — reverses a boolean

14. Loops Overview

for (let i = 1; i <= 10; i++) {
  console.log(i);
}

15. Nested for Loops (Multiplication Table)

for (let i = 2; i <= 9; i++) {
  document.write(`<h3>${i}단</h3>`);
  for (let j = 1; j <= 9; j++) {
    document.write(`${i} × ${j} = ${i * j}<br>`);
  }
}

16. while and do...while Loops

let n = 5;
while (n > 0) {
  document.write("*");
  n--;
}

do {
  document.write("*");
  n--;
} while (n > 0);

17. break and continue

for (let i = 1; i <= 10; i++) {
  if (i === 5) break; // stop loop
}
